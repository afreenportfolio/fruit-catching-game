# Fruit Catching Game
This PyGame-based application simulates a fruit-catching challenge with dynamic elements like falling objects and player controls. It includes custom-designed UI assets for visuals, ensuring an engaging and responsive gaming experience.

Features
1. Real-time event handling and collision detection.
2. Randomized fruit generation and scoring.
3. Custom icons, screens, and graphics.
4. Unit testing for gameplay reliability.

Technologies Used
1. Language: Python
2. Library: PyGame
3. Tools: VS Code

Installation
1. Clone the repository: git clone https://github.com/afreenportfolio/fruit-catching-game.git
2. Install dependencies: pip install pygame

Usage
1. Run the game: python fruit_catching_game.py
2. Use arrow keys to move and catch fruits.
3. Aim for high scores!
